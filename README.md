# Sales POS System

Sistema de punto de venta desarrollado en Java con integración a Supabase para gestión de productos y ventas.

## Tecnologías
- Java
- Supabase (API REST)
- JSON
- Git

## Funcionalidades
- Gestión de productos
- Registro de ventas
- Cálculo automático de totales
- Manejo básico de inventario
- Integración con base de datos en la nube
